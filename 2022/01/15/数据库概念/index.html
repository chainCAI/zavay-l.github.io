数据库

  • 存储数据的仓库 , 数据是有组织的进行存储 .

  • DataBase 简称DB

    数据库管理系统

  • 管理数据库的大型软件

  • DataBase Management System 简称DBMS

    SQL

  • Structured Query Language 简称SQL 结构化查询语言

  • 操作关系型数据库的编程语言

  • 定义操作所有关系型数据库的统一标准

常见的关系型数据库管理系统

  • Oracle：收费的大型数据库，Oracle 公司的产品
  • ==MySQL==： 开源免费的中小型数据库。后来 Sun公司收购了 MySQL，而 Sun 公司又被 Oracle 收购
  • SQL Server：MicroSoft 公司收费的中型的数据库。C#、.net 等语言常使用
  • PostgreSQL：开源免费中小型的数据库
  • DB2：IBM 公司的大型收费数据库产品
  • SQLite：嵌入式的微型数据库。如：作为 Android 内置数据库
  • MariaDB：开源免费中小型的数据库

安装配置

解压后将软件文件目录放入exe专属软件安装路径
1.开始配置环境
配置环境变量
MYSQL_HOME D:\exe\mysql-5.7.24-winx64
Path %MYSQL_HOME%\bin
2.新建配置文件 my.ini 到MySQL安装路径下

3.打开管理员权限的命令提示符
4.初始化MySQL
mysqld --initialize-insecure
就会在MySQL安装路径下生成一个data文件夹及其内容
5.注册MySQL服务
mysqld -install
可以查看后台服务services.msc会发现存在MySQL的服务已经建立
6.启动MYSQL服务
net start mysql//启动MySQL服务
net stop mysql //停止MySQL服务
在启动MySQL服务的时候 我出现了问题 发生系统错误2 系统找不到指定的文件
解决方法: 将命令路径cd到MySQL安装路径的bin目录下 再使用命令
7.修改默认账户 密码
mysqladmin -u root password 1234
mysql管理员 -管理者 root(用户名) 密码 1234

使用

登录和登出
mysql -uroot -p1234//登录
登录参数
mysql -u用户名 -p密码 -h要连接的mysql服务器的ip地址(默认127.0.0.1) -P端口号(默认3306)
退出
exit
quit

关系型数据库

关系型数据库是建立在关系模型基础上的数据库，简单说，关系型数据库是由多张能互相连接的 二维表 组成的数据库
展示出来是一种表格的形式

最最简单的例子

订单信息
订单编号 负责人 日期
34535 露露 2022年5月3日
24235 天天 2022年5月4日
54654 苗苗 2022年5月4日
37564 海海 2022年5月4日
12342 达达 2022年5月4日

关系型数据库的优点

  • 都是使用表结构，格式一致，易于维护。
  • 使用通用的 SQL 语言操作，使用方便，可用于复杂查询。
    • 关系型数据库都可以通过SQL进行操作，所以使用方便。
    • 复杂查询。现在需要查询001号订单数据，我们可以看到该订单是1号客户的订单，而1号订单是李聪这个客户。以后也可以在一张表中进行统计分析等操作。
  • 数据存储在磁盘中，安全。

数据模型


通过客户端可以通过数据库管理系统创建数据库，在数据库中创建表，在表中添加数据。
创建的每一个数据库对应到磁盘上都是一个文件夹。create database db1;

db.frm 是表文件
db.MYD 是数据文件
通过这两个文件就可以查询到数据展示成二维表的效果。

总和

  • MySQL中可以创建多个数据库，每个数据库对应到磁盘上的一个文件夹
  • 在每个数据库中可以创建多个表，每张都对应到磁盘上一个 frm 文件
  • 每张表可以存储多条数据，数据会被存储到磁盘中 MYD 文件中
